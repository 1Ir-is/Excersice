        Thực hiện kiểm tra.								
        + Nếu từng phần tử ở vị trí 1/2 mảng sau có giá trị bằng bình phương từng phần tử đầu thì trả về "true",									
        + Nếu bất kỳ vị trí nào không đúng điều kiện thì trả về "false"								
        + Nếu mảng có số phần tử lẻ => false									
        + Ví dụ: checkArray(1,2,3,1,4,9]) => true = (vì 1*1=1 và 2*2=4 và 3*3=9)									
                checkArray([2,1,3,1,1,9]) => false = (vì 2*2=4  khác 1 => sai)
